Arcader

  • Object Oriented Javscript
  • HTML Canvas

Get Coffeed

Get Coffeed is a project that utilizes the MVC framework Knockout.js and two APIs - Google Maps and Foursquare.

Fractal Dance

Fractal dance is built from the P5 Javascript engine. I built this project for Two Sigma's Math Hackathon at MoMath. The project would be loaded onto a pressure sensitive screen that people can interact with. The background color and fractal length changes according the the number of people standing on the screen.

Tech Stack
  • P5.js
  • Javascript
  • MathSquare SDK
